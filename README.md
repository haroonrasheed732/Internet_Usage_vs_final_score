# 📊 Simple Linear Regression: Internet_Usage → Final_Score

## 📌 Overview

This project analyzes the relationship between **Internet Usage** and **Final Score** using Simple Linear Regression.

---

## 🎯 Objective

* Predict Final Score using Internet Usage
* Understand its impact on student performance

---

## 🧾 Feature & Target

* Input Feature (X): Internet_Usage
* Target Variable (y): Final_Score

---

## 🔧 Data Preprocessing

* Missing values handled
* Duplicate rows removed
* Categorical encoding applied
* Feature scaling performed

---

## ⚙️ Model Training

* Algorithm: Linear Regression
* Data split: 80% train / 20% test

---

## 📈 Model Evaluation

* Metric: R² Score

---

## 📊 Visualization

* Scatter plot + regression line

📌 Insight:

* Higher internet usage may negatively impact academic performance

---

## 💾 Model Saving

* Saved using joblib

---

## 🧠 Conclusion

* Excessive internet usage can reduce study efficiency and scores
